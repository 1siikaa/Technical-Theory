What is the difference between ai/ml/deep learning?
AI: Any system that behaves intelligently. AI make machines act smart. Doesn’t always need learning. Any Simple Rule Based System.
Examples: Rule-based chatbot (“if user says hi → reply hello”), hess-playing program

ML: Learning from existing data is ML. Subset of AI (Bigger Picture). Instead of writing rules, you train using data. System finds patterns on its own.
Examples: Spam detection (learns from emails), Netflix recommendations.

DL: Subset of ML. Learning like a brain (neural networks). Uses neural networks (complex models) Works well with large data
Examples:
	Face recognition
	ChatGPT
	Voice assistants
